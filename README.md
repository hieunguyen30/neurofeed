 # Neurofeed 🎙️

A podcast recommendation app that detects your mood using AI and recommends podcasts accordingly.

## Tech Stack
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- AI: Claude API (mood detection)
- Database: Supabase (Postgres)
- Podcasts: Podcast Index API
